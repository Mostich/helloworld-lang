# helloworld-lang

The HelloWorld Programming Language

Programming language for output Hello World

File extension: .hw

Example:
```nim
msg -> print
echo -> print

var hello = "Hello, World!"

echo: ("Hello, World!")
print ("Hello World")
msg: (hello)
```
```bash
$helloworld example.hw
$helloworld -f example.hw
$helloworld example.hw -a
$helloworld example.hw -i
$helloworld --help
```
